## 2026-06-19 상태 갱신: 검색 결과 내보내기

- 필터된 장비 목록을 팀원에게 전달할 수 있도록 결과 내보내기 기능을 추가했다.
- 검색 패널에서 `결과 요약 복사`와 `CSV 다운로드`를 제공한다.
- CSV는 현재 필터 결과 기준으로 장비명, 분류, 국가, 원산국, 제조사, 운용 상태, 임무 태그, 계열 수, 출처 신뢰도, 상세 ID를 포함한다.
- CSV 파일명은 `equipment-search-results.csv`로 고정해 반복 공유와 저장이 쉽도록 했다.

## 2026-06-19 상태 갱신: 검색 조건 공유 링크

- 장비 검색 화면의 필터 상태가 URL query string에 동기화된다.
- 공유 가능한 검색 조건은 `q`, `family`, `category`, `role`, `country`, `status`, `maturity`, `eq` 파라미터로 표현한다.
- 검색 링크를 새로고침하거나 팀원에게 전달해도 같은 필터 결과가 복원된다.
- 검색 패널에 `검색 링크 복사` 버튼을 추가했다.

## 2026-06-19 상태 갱신: 팀원용 검색 필터 고도화

- 장비 검색 화면에 세부 필터를 추가했다.
- 필터 축은 `차륜/궤도 계열`, `장비 분류`, `임무/파생형`, `국가`, `운용 상태`, `계열 성숙도`, `자유 검색어`로 구성한다.
- 기존 JSON 스키마는 유지하고 `roleTags`, `operatorCountries`, `status`, `variants.role`, `variants.maturity`에서 필터 옵션을 자동 파생한다.
- 검색 결과 행에 역할 태그와 계열 수를 표시해 팀원이 상세 페이지로 들어가기 전 기본 판단을 할 수 있게 했다.
- 3D는 계속 추후 기능으로 유지하고 현재 화면은 검색/상세/출처 중심이다.

## 2026-06-19 상태 갱신

- 현재 제품 방향을 `전 세계 지상 장비 검색 카탈로그`로 조정했다.
- 첫 화면은 장비 검색, 분류 필터, 지도, 선택 장비 요약, 상세 정보로 구성한다.
- 상단 메뉴는 `장비 검색 / 전체 장비 / 전장 인사이트 / 출처`로 단순화했다.
- 3D는 현재 핵심 기능에서 제외하고, 추후 GLB 파일을 연결할 수 있는 준비 슬롯과 공개 부품 스펙 목록으로 축소했다.
- `Compare`, `Technologies`, `Cases`, `Development`의 기존 독립 메뉴 역할은 검색 중심 흐름에 맞춰 흡수했다.
- 기존 데이터는 삭제하지 않고 장비 상세, 전장 인사이트, 출처 인덱스에서 계속 활용한다.
